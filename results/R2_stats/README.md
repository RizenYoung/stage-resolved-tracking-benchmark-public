# R2 Public Aggregate Audit Tables

This directory contains selected public-safe aggregate tables from the Stage-II handoff and fitting audit.

Included tables:

- `R2_covariance_audit_extract_2026-06-10.csv`
- `R2_extreme_ablation_extract_2026-06-10.csv`
- `R2_kappa_observability_extract_2026-06-10.csv`
- `R2_paired_posthoc_stats_2026-06-10.csv`
- `R2_tail_normality_laplace_2026-06-10.csv`

The retained files support the manuscript's limited claims about aggregate calibration improvement, non-Gaussian boundary behavior, and weak-observability regimes. Tables carrying local path fields, restricted source identifiers, or detailed track-level exports are not included in this public package.

Related figures are stored in `figures/R2_stats/`.
