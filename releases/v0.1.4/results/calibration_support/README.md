# Calibration Supporting Result Tables

This directory contains aggregate calibration tables supporting the Stage-II discussion.

Included tables:

- `csv/charge_flip_summary.csv`
- `csv/pull_mechanism.csv`
- `csv/pull_policy_summary.csv`
- `csv/top_tail_profile.csv`
- `csv/outlier_screening.csv`
- `csv/pit_coverage.csv`
- `csv/tail_family_comparison.csv`
- `csv/coverage_scale_scan.csv`
- `csv/coverage_scale_summary.csv`
- `csv/increment_correlation.csv`

Interpretation summary:

- Pull widening is the dominant curvilinear-pull mechanism in the tested sample; bias dominance is not supported.
- Top pull tails are not confined to the most extreme transport-invalid regime.
- PIT and coverage checks show conservative behavior with shape departures.
- Tail-family behavior is stratum-specific rather than universal.
- The scale scan does not support a universal 0.80 rescaling recommendation.

The symbolic covariance note is stored as `increment_covariance_note.md`. Related figures are stored in `figures/calibration_support/`.
