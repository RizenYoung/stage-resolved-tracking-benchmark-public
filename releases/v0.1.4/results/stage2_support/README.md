# Stage-II Supporting Result Tables

This directory contains aggregate tables supporting the Stage-II handoff and fitting results.

Included tables:

- `covariance_summary.csv`
- `extreme_regime_summary.csv`
- `condition_observability_summary.csv`
- `paired_calibration_statistics.csv`
- `tail_distribution_summary.csv`

The tables support the manuscript's bounded claims about aggregate calibration improvement, non-Gaussian boundary behavior, and weak-observability regimes.

Related figures are stored in `figures/stage2_support/`.
