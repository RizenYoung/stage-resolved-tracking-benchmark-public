# Data Access Note

## Included Materials

- Aggregate result tables supporting manuscript-level diagnostics.
- Final PDF and PNG figure assets used in the manuscript.
- Supporting calibration figures and aggregate tables.
- Documentation, citation metadata, licenses, and package-validation utilities.

## Materials Not Distributed

The original event generator, canonical `SiHits_3D` benchmark dataset, derivative source-level event and track data, and trained model weights are not distributed in this package. Permission for their public redistribution has not been confirmed.

## Interpretation

The released materials support inspection of aggregate results and figures reported in the manuscript. They are not intended to reproduce the complete source-data workflow.
